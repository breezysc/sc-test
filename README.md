# AutoBuy 自动购买工具

自动购买游戏内物品并自动存仓的辅助工具。

## 功能特性

- 🎯 **自动购买**: 基于图像识别自动识别并购买高亮物品
- 📦 **智能存仓**: 自动检测背包格子，智能执行存仓操作
- 🗄️ **仓库管理**: 支持12x12自有仓库，自动检测容量并切换标签页
- 📱 **窗口追踪**: 支持游戏窗口移动后自动校准坐标
- 🖥️ **UI启动器**: 提供图形界面配置启动参数

## 快速开始

### 运行方式

1. **使用UI启动器（推荐）**:
   ```
   python launcher.py
   ```
   或运行打包后的 `AutoBuy.exe`

2. **命令行方式**:
   ```
   python auto_buy_new.py --server=china
   ```

### 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--server` | 服务器类型: `china` 或 `global` | `china` |
| `--test-stash` | 测试模式，直接执行存仓 | 否 |
| `--debug` | 调试模式，输出详细日志 | 否 |
| `--config` | 指定配置文件路径 | 无 |

## 配置说明

### 坐标配置

工具使用相对游戏窗口的坐标系统，支持窗口移动后自动校准：

- **背包格子**: 12列 × 5行 = 60个格子
- **自有仓库**: 12列 × 12行 = 144个格子
- **仓库标签页**: 自动检测并切换

### 阈值配置

| 参数 | 说明 | 值 |
|------|------|-----|
| `EMPTY_CELL_THRESHOLD` | 空格子判定阈值 | 0.9 |
| `STASH_FULL_THRESHOLD` | 仓库满仓阈值 | 80% |
| `TAB_THRESHOLD` | 标签页匹配阈值 | 0.45 |

## 文件结构

```
├── AutoBuy.exe          # 打包后的可执行文件
├── launcher.py          # UI启动器
├── auto_buy_new.py      # 主程序
├── hsv_detector.py      # HSV检测器
├── window_locator.py    # 窗口定位器
├── template_locator_integrator.py  # 模板定位集成器
├── cc.png               # 空格子参考模板
├── auto_buy_config.json # 配置文件
├── good.json            # 物品配置
├── templates/           # 模板图片目录
│   └── stash/           # 仓库模板
└── stash_debug/         # 调试截图目录
```

## 操作流程

1. 启动工具，选择服务器类型（默认国服）
2. 工具自动检测游戏窗口
3. 检测到高亮物品时自动执行购买
4. 购买后按ESC关闭对话框
5. 打开仓库后检查容量
6. 满仓时自动切换到下一个标签页
7. 执行智能存仓操作

## 技术说明

- **语言**: Python 3.8+
- **GUI**: Tkinter
- **图像识别**: OpenCV + mss
- **自动化**: PyAutoGUI
- **打包**: PyInstaller

## 运行要求

- Windows 10/11
- 游戏窗口需在前台可见
- 推荐分辨率: 1920×1080

## 注意事项

1. 使用前请确保游戏窗口已打开
2. 工具运行期间请保持游戏窗口在前台
3. 建议在游戏设置中关闭全屏模式
4. 首次使用可能需要调整坐标配置

## 开发

### 打包

```
python build.py
```

打包后生成 `dist/AutoBuy.exe`。
